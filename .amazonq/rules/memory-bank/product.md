# Content Analyzer - Product Overview

## Project Purpose
Intelligent content analysis agent that extracts, analyzes, and evaluates web content using AI. This is a TypeScript version converted from Python (Streamlit + FastAPI) to modern web technologies.

## Value Proposition
- Automated content extraction from any URL
- AI-powered text analysis using Gemini/Groq APIs
- Source credibility evaluation
- Code detection and analysis within content
- Additional reference discovery
- Complete Arabic interface support
